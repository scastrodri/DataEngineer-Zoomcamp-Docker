Question 4. Which was the pick up day with the longest trip distance? Only consider trips with trip_distance less than 100 miles.
Answer: Use query 
select lpep_pickup_datetime, trip_distance from public.yellow_taxi_data_trips_2025_11
where trip_distance < 100
order by trip_distance desc
limit 1;
The results shows "2025-11-14 15:36:27"	88.03

Question 5. Which was the pickup zone with the largest total_amount (sum of all trips) on November 18th, 2025?
Answer: Use query
SELECT 
    tz."Zone", 
    SUM(yt."total_amount") AS daily_total_amount
FROM public.yellow_taxi_data_trips_2025_11 AS yt
INNER JOIN public.taxi_zone_lookup AS tz
    ON yt."PULocationID" = tz."LocationID"
WHERE yt."lpep_pickup_datetime" >= '2025-11-18 00:00:00' 
  AND yt."lpep_pickup_datetime" <= '2025-11-18 23:59:59'
GROUP BY tz."Zone"
ORDER BY daily_total_amount DESC
LIMIT 1;
The results shows "East Harlem North"	9281.919999999996

Question 6. For the passengers picked up in the zone named "East Harlem North" in November 2025, which was the drop off zone that had the largest tip?
Answer: Use query
SELECT 
    tz_do."Zone" AS dropoff_zone, 
    MAX(yt."tip_amount") AS largest_tip
FROM public.yellow_taxi_data_trips_2025_11 AS yt
INNER JOIN public.taxi_zone_lookup AS tz_pu
    ON yt."PULocationID" = tz_pu."LocationID"
INNER JOIN public.taxi_zone_lookup AS tz_do
    ON yt."DOLocationID" = tz_do."LocationID"
WHERE tz_pu."Zone" = 'East Harlem North'
  AND yt."lpep_pickup_datetime" >= '2025-11-01 00:00:00'
  AND yt."lpep_pickup_datetime" < '2025-12-01 00:00:00'
GROUP BY tz_do."Zone"
ORDER BY largest_tip DESC
LIMIT 1;
The results shows "Yorkville West"	81.89
